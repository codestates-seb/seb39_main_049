## 👏 소개
> 안녕하세요! 취업 및 이직을 준비하는 개발자들을 위한 면접 커뮤니티, **세이뷰** 입니다!🐳

'SAVIEW'는 'SAVE'와 'INTERVIEW'의 합성어로 취업을 위해 면접을 준비하는 개발자들을 위해 면접질문과 답변을 저장해놓은 공간을 의미합니다.  
개발 면접 질문에 대한 양질의 답변을 얻고 그것들을 비교해가며 생각을 정립하는 것에 대한 어려움 해소하고자 시작한 프로젝트 입니다.
